Приложение содержит более 100 экранов: сплеш-экран, авторизация, главная страница, каталог товаров, поиск, корзина, профиль пользователя, оплата, кошелек, отслеживание заказов и история покупок. Все интерфейсы поддерживают светлую и темную темы. Бета-версия уже готова к тестированию и интеграции с различными backend-решениями.

Стандартные индикаторы загрузки во Flutter выглядят скучно, особенно в e-commerce приложениях. Мы создали кастомные анимации, которые удерживают внимание пользователя даже во время загрузки контента — как в популярных приложениях YouTube и LinkedIn. Это небольшая, но важная деталь для улучшения пользовательского опыта.

## Экраны в бета-версии FlutterShop

Приложение включает более 100 тщательно проработанных экранов для всех сценариев использования:

### Онбординг
- Выбор категории товаров
- Добавление в корзину
- Онлайн-оплата
- Отслеживание заказа
- Поиск магазинов
- Настройка уведомлений

### Аутентификация
- Вход в аккаунт
- Восстановление пароля
- Верификация кодом
- Создание нового пароля
- Регистрация
- Настройка профиля
- Условия использования
- Биометрическая аутентификация

### Товары
- Страница продукта
- Уведомления о наличии
- Мгновенная покупка
- Детальное описание
- Отзывы пользователей
- Добавление отзыва
- Способы доставки
- Политика возврата
- Таблица размеров

### Главная страница
- Домашний экран
- Распродажи
- Детские товары
- Бренды
- Каталог категорий
- Избранные товары

### Поиск
- История поиска
- Подсказки
- Фильтры (размер, цвет, бренд, цена)
- Сортировка результатов

### Корзина
- Список товаров
- Пустая корзина
- Выбор адреса доставки
- Подтверждение заказа
- Способы оплаты
- Добавление новой карты

### Профиль
- Личный кабинет
- Уведомления
- Язык интерфейса
- Адреса доставки
- Платежные методы
- Кошелек
- История транзакций
- Техподдержка

### Заказы
- Текущие заказы
- Отмененные заказы
- Доставленные заказы
- Возвраты

### Дополнительно
- Обработка ошибок
- Запросы разрешений
- Состояния "пустых" экранов
- Оффлайн-режим
